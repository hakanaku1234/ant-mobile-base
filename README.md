# React+ant-mobile+graphQLServer 服务端渲染 
优化打包、服务端首屏渲染、按需加载、按屏幕展示加载组件、使用antd-moubileUI框架、webpack4


* 客户端使用React + Redux + Ant-Mobile
* 服务端使用Koa2 + GraphQL 

### 启动

``` 
//安装依赖
npm install

//服务器环境
npm run build
npm run server

//开发环境
npm run start

```
